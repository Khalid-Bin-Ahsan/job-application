1. getElementById() = Selects one element by its ID.
getElementsByClassName() = Selects elements by class name and Returns a live collection.
querySelector() = Selects the first matching element using CSS selector.
querySelectorAll() = Selects all matching elements using CSS selector. Returns a static NodeList.


2.Create element using document.createElement()
Add content using textContent or innerHTML
Insert using appendChild() or append()


3.Event Bubbling is when an event starts from the target element and moves upward to its parent elements.


4. Event Delegation means adding an event listener to a parent element instead of multiple child elements.


5.preventDefault() = Stops the default browser action.
stopPropagation() = Stops the event from moving to parent elements.
